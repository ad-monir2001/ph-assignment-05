Important FAQ Section

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a webpage as a tree of objects, where each object corresponds to a part of the document (like elements, attributes, or text).

How do you select an element from the DOM?

To select an element from the DOM (Document Object Model) in JavaScript, I can use various methods depending on what I'am looking for. Here are the most common ways:
document.getElementById()
document.getElementsByClassName()
document.getElementsByTagName()
document.querySelector()

What is event delegation in the context of the DOM, and why is it useful?

DOM event delegation is a technique where I attach a single event listener to a parent element, instead of adding separate listeners to individual child elements. When an event occurs, the event is propagated up to the parent element, where I can handle the event for specific child elements.
This is useful for dynamically created elements.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an elements we use getAttribute(), setAttribute() property.
To styles using DOM we use directly style property. Ex: element.style.color = 'red';
and classList methods: Add, remove.
