Important FAQ Section

What is the DOM?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a webpage as a tree of objects, where each object corresponds to a part of the document (like elements, attributes, or text). The DOM allows scripts to dynamically access and manipulate the content, structure, and styling of the webpage.

What is the difference between addEventListener and onclick?

addEventListener allows you to attach multiple event listeners to an element for the same event, and it provides more control, like specifying whether the event should capture or bubble.
onclick assigns a single event handler for the click event. If another function is assigned to onclick, the previous one is overwritten.

What is DOM event delegation?

DOM event delegation is a technique where you attach a single event listener to a parent element, instead of adding separate listeners to individual child elements. When an event occurs, the event is propagated up to the parent element, where you can handle the event for specific child elements. This is useful for dynamically created elements.

How does DOM traversal work?

DOM traversal refers to navigating between nodes in the DOM tree. You can move between parent, child, and sibling elements using various properties:
parentNode: Accesses the parent of a node.
firstChild, lastChild: Access the first and last child of a node.
nextSibling, previousSibling: Move to the next or previous sibling of a node.
children: Gets all child elements of a node.

How do you remove an element from the DOM?

You can remove an element from the DOM using one of the following
methods: element.remove(): Removes the element from its parent.
parentNode.removeChild(child): Removes the child node from the parent element.
